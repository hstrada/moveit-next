## 📦️ How to install the project

To clone the project, use the commands bellow:

```bash
  # Clone the repository
  ❯ git clone https://github.com/hstrada/moveit-next.git

  # Enter directory
  ❯ cd moveit-next
```

### FrontEnd

**Emulate in IOS**

```bash
  # Enter frontend directory
  > cd moveit-next

  # Install the dependencies
  ❯ yarn

  # Start the project
  ❯ yarn ios
```
